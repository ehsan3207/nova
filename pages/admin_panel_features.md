#  مدیریت کاربران

###  بخش ها:
- جست و جو کابران  
- فیلتر نقش  
- افزودن کاربر  
- لیست کاربران  

###  افزودن کاربر:
- نام و نام خانوادگی
- شماره تماس
- ایمیل
- جنسیت
- نقش (مدیر مالی / مدیر فنی)


###  لیست کاربران:
- نام و نام خانوادگی
- شماره تماس
- ایمیل
- مقطع
- پایه
- رشته
- جنسیت
- استان
- وضعیت
- نقش
- عملیات (ویرایش و حذف کاربر)
---

#  تیکت و پشتیبانی

### بخش ها:
- فیلتر اولویت ها ، وضعیت و نوع تیکت 
- لیست تیکت ها    

###  لیست تیکت ها:
- شماره تماس
- موضوع
- نوع اولویت (زیاد / متوسط / کم)
- نوع وضعیت (در انتظار / پاسخ داده شده)
- نوع تیکت (فنی / مالی)
- تاریخ ایجاد
- عملیات (مشاهده / حذف)

---
#  اعلانات

### بخش ها:
- افزودن اعلانات (دریافت متن)
- لیست اعلانات    

###  لیست اعلانات:
- متن اعلان
- تاریخ ایجاد
- عملیات حذف
